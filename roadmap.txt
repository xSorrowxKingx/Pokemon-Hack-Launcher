## Roadmap

Planned features and improvements for future versions of the launcher:

- **Improved visual design**
  - Add a more polished background to make the launcher visually appealing.
  - Improve hover animations for buttons and UI elements.

- **Game icons and visual elements**
  - Display a small icon next to each game / hack in the launcher.
  - Optionally show a banner or header image for the selected game.

- **Categories / organization**
  - Add categories to organize Pokémon fan games and ROM hacks more clearly.

- **Favorites and recently played**
  - Allow users to mark games as favorites.
  - Show a list of recently launched games for quick access.

- **Keyboard controls**
  - Allow pressing **Enter** to launch the currently selected game.

- **Performance improvements**
  - Reduce launcher startup time and improve overall loading speed.

- **Windows build**
  - Provide a compiled `.exe` version of the launcher for easier distribution.
